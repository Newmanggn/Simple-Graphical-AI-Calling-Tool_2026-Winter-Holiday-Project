# 简单图形化AI调用工具

一个使用Python+Flask开发的图形化AI模型调用和流程编排工具，通过拖拽式模块配置、连线式流程定义，让AI应用开发更简单！

## 功能特性

- 🎨 **拖拽式模块配置** - 通过拖拽方式设置模块，每个模块有输入输出接口
- 🔗 **连线式流程定义** - 通过连线设定运行顺序、测试和数据流向
- 📝 **一键代码转换** - 将图形化编程结果一键转换为Python代码
- 🧩 **自定义模块支持** - 支持自定义模块并配有图形化的编程工具
- 📊 **运行状态可视化** - 连线上自动标出运行步骤，黄色圆圈显示当前运行状态
- 🎯 **多种模块类型** - 输入模块、调用模块、处理模块、输出模块等
- ⚡ **步进/连续运行** - 支持步进运行和连续运行两种模式
- 🔴⚫ **连线颜色切换** - 用户可通过双击连线切换红黑色
- 💾 **项目保存/加载** - 支持保存和加载`.aiud`格式的项目文件

## 技术栈

- **后端**: Python + Flask
- **前端**: HTML/JavaScript

## 安装与运行

### 环境要求

- Python 3.8+

### 快速开始

1. 克隆或下载本项目到本地

2. 安装依赖：
```bash
pip install flask
```

3. 运行应用：
```bash
python app.py
```
或者在Windows上直接双击运行 `run_app.cmd`

4. 在浏览器中访问：`http://localhost:5000`

## 使用说明

### 基本操作流程

1. **添加模块** - 从左侧模块库拖拽模块到中央画布
2. **配置模块** - 双击模块可打开配置面板
3. **连接模块** - 点击模块输出端口，拖动到目标模块输入端口建立连接
4. **运行流程** - 点击顶部控制栏的运行按钮开始执行
5. **保存项目** - 点击保存按钮，项目会自动保存为`.aiud`文件

### 模块类型

- **输入模块** - 负责数据输入，包括用户输入、固定值输出等
- **处理模块** - 负责数据处理、转换和条件判断
- **调用模块** - 负责调用AI模型或其他外部服务
- **输出模块** - 负责结果输出和展示

### 连线说明

- **黑线** - 数据传递，仅在目标模块`excitedbydata=true`时继续传递
- **红线** - 立即激活目标模块执行
- 双击连线可切换颜色
